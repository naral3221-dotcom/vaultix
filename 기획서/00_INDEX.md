## 5. 사용 안내 (개발 시작 시)

1. `00_IMPLEMENTATION_SPEC_v0.4.md`를 먼저 읽어 MVP 구현 범위와 폐기 결정을 고정한다.
2. `00_확정사항_레지스트리.md`를 읽어 모든 B/T/D 결정사항을 확인한다.
3. Phase 0~1은 `00_IMPLEMENTATION_SPEC_v0.4.md`의 Sprint 순서와 `01/02` 문서의 WBS를 함께 사용한다.
4. MVP 완료 후에는 `07_PostMVP_배포전략.md`의 Release 1부터 순차 배포한다.
5. 각 단계 완료 시 체크박스 표시 + 변경사항은 레지스트리에 추가 기록한다.
6. 막히는 부분은 코워크에 "Phase X §Y 항목에서 막힘" 형식으로 질문한다.
